# wasd-editor
A terminal text editor for the minimalist gamer.

### Goals
1. Fun
2. Efficient
3. Discoverable
4. Consistent (To a fault?)
5. One way to do things

### Features
- Only 1 selection
- Replace (all)
- Search
- Shift extend
- 2 modes with 2 permanent buttons
- move line up/down (treesitter incl?)
- Start/end of line
